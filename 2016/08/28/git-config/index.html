hexo是一款基于Node.js的静态博客框架，可以方便地生成静态网页，其官网：https://hexo.io/
基于hexo，再利用github提供的免费博客服务github pages，我们可以方便地搭建一个属于自己的博客网站。

若对Node.js本身有兴趣，推荐：http://www.runoob.com/nodejs/nodejs-tutorial.html。
